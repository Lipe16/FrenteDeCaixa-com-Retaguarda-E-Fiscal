Aqui disponibilizarei todo material de pesquisa envolvido neste projeto, inclusive um projeto de NFEeNFCE pronto com Java e interfaces Gráfica no swing... Esse projeto vai me ajudar muito na construção do sistema de PDV e Admin para lojas.


Ao abrir o projeto "NFE_NFCE_CursoNeri", precisarei de usar a IDE NetBeans e o Banco de dados Será PostGre... As configurações podem ser alteradas no arquivo "nfe" encontrado no diretorio "MaterialDePesquisa", inclusive essa pasta "nfe" ficará na raiz do meu disco "C:/" (uso Windows).
